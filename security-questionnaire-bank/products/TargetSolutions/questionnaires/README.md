# TargetSolutions — Completed Questionnaires

Answered questionnaires for TargetSolutions are stored here after processing.

## Naming Convention
`YYYY-MM-DD-client-name-description.ext`

Examples:
- `2026-02-26-fire-dept-security-assessment.xlsx`
- `2026-03-15-county-ems-vendor-review.pdf`
- `2026-04-01-state-agency-compliance-questionnaire.docx`

## Flow
1. New questionnaire dropped in `imports/`
2. Claude Code answers it using the knowledge base
3. Completed file moved here automatically
