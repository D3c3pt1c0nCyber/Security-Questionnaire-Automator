# Past Questionnaires

This folder stores completed questionnaire files for reference.

## Naming Convention
Use this format: `YYYY-MM-client-name-description.ext`

Examples:
- `2025-06-acme-corp-vendor-assessment.xlsx`
- `2025-09-bigbank-security-questionnaire.pdf`
- `2026-01-techco-due-diligence.docx`

## Usage
These files serve as reference material. When answering new questionnaires,
Claude Code can search these for tone, detail level, and organization-specific context.
