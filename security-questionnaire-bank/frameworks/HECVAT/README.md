# HECVAT — Higher Education Community Vendor Assessment Toolkit

## Version History

### HECVAT v4 (Latest — Feb 2025)
- Merged Full, Lite, Triage, and On-Premise into **one unified toolkit**
- Added new Privacy and AI questions
- New documentation for 3 personas (evaluators, service providers, campus communities)
- Download from: https://www.educause.edu/higher-education-community-vendor-assessment-toolkit
  (requires free EDUCAUSE account)

### HECVAT v3 (Previous — 3.06 was the last v3 release)
- **Full** (v3.06) — Comprehensive ~250+ questions ✅ Template downloaded
- **Lite** (v2.11) — Abbreviated ~100 questions ✅ Template downloaded
- **Triage** (v2.10) — Quick screening tool ✅ Template downloaded
- **On-Premise** — For self-hosted deployments

## Folder Structure
```
HECVAT/
├── v4/
│   ├── templates/          ← Download HECVAT 4 from EDUCAUSE and place here
│   └── completed/          ← Answered HECVAT v4 questionnaires
└── v3/
    ├── templates/           ← ✅ Full v3.06, Lite v2.11, Triage v2.10 downloaded
    └── completed/
        ├── Full/            ← Answered HECVAT Full questionnaires
        └── Lite/            ← Answered HECVAT Lite questionnaires
```

## Naming Convention
`YYYY-MM-DD-[ProductName]-HECVAT-[version].ext`

Examples:
- `v4/completed/2026-02-26-SafeLMS-HECVAT-v4.xlsx`
- `v3/completed/Full/2025-06-15-TargetSolutions-HECVAT-Full-v3.06.xlsx`
