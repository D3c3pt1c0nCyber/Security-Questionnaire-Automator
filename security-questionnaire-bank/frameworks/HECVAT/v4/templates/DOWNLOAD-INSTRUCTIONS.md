# Download HECVAT v4 Template

HECVAT v4 requires a free EDUCAUSE account to download.

## Steps
1. Go to https://www.educause.edu/higher-education-community-vendor-assessment-toolkit
2. Sign in or create a free EDUCAUSE account
3. Download the HECVAT 4 toolkit (.xlsx)
4. Place the file in this folder

## What's in HECVAT v4
- Unified toolkit (replaces Full, Lite, Triage, On-Premise)
- New Privacy questions
- New AI-specific questions
- Training materials for evaluators and service providers
