Atlassian Authentication

One set of credentials for both Confluence and Jira

Your Atlassian Cloud instance URL (used for both Confluence and Jira)
Your Atlassian account email address
Create an API token at id.atlassian.com/manage-profile/security/api-tokens
Jira project key for security questionnaire tickets (e.g. ISC, SEC, SECQ)
How to get API credentials
  1. Go to Atlassian API Tokens
  2. Click "Create API token" and give it a label
  3. Copy the token and paste it above
  4. Use your Atlassian account email as the email field
  5. Both Confluence and Jira use the same credentials

Note: Credentials are stored in server memory and .env file
